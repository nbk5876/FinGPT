FinGPT Prototype

March 3 2024 - The prototype accepts queries about an equity and returns stock market information and analysis.

Github Project Folder

Processing Steps

  • Step 1 - Get natural language query from the user. For example, "What is the Price-to-Earnings ratio for Microsoft?"
  • Step 2- Fetch current financial data from the Web: The system retrieves the latest data from the internet using APIs or scraping techniques. (Example API Return Value)
  • Step 3- Data structuring and, optional, lightweight local data caching: This is an optional step where the system may cache the retrieved data locally to improve performance and reduce the need for repeated data fetching. Maybe implement this in a later phase.
  • Step 4 - Resolve NLP (natural language processing) query using OpenAI API's:  The system sends the user's query together with retrieved financial data to the OpenAI API, which processes the query using the GPT LLM model.
  • Step 5 - Send results back to the user: Finally, the system presents the OpenAI API's response back to the user.

The average P/E ratio tends to hover around 20-25 for the companies in the S&P 500, so in comparison Ford's P/E ratio might suggest it is undervalued, assuming all other factors being equal.
